# Cups and Balls VR - Unity 6 Projekt

Multiplayer VR igra "Čaše i Loptice" sa XR Interaction Toolkit-om.

## 🚀 Brzi Start

```bash
git clone https://github.com/radovanovicema/cups-and-balls-vr.git
cd cups-and-balls-vr
```

Otvori u **Unity 6000.0.64f1** i pritiski **Play**!

---

## 📋 Šta je Uključeno

✅ **Kompletan GameManager** - Upravljanje igrom (faze, score, logika)
✅ **5 Interaktivnih Čaša** - Sa XR Grabovanjem
✅ **2 Nasumične Loptice** - Sa fizikom
✅ **UI Sistem** - Poruke, pokušaji, restart dugme
✅ **Audio Feedback** - Zvukovi za svaku akciju
✅ **Mešanje Čaša** - Animirane zamene pozicija
✅ **Auto Setup** - Sve konfigurira automatski

---

## 🎮 Kako Igrati

1. Počnite igru
2. Gledajte gde su loptice (3 sekunde)
3. Čaše se mešaju (5 sekundi)
4. Grabujte čaše sa 3 pokušaja
5. Pronađite obe loptice i pobedite! 🎉
